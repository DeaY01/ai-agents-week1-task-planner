# Week 1 – Basic Task Planner Agent  
**12-Week AI Agents Series • December 2025**

The simplest, most powerful agent you can run **today** – no code, no API keys, no install.

### 10-second demo
You type:  
> Plan a cheap 3-day weekend trip to Abeokuta from Lagos in January, max #200,000 total.

Agent instantly returns a perfect, executable 7-step plan → ready for full autonomy in later weeks.

### How to run it right now
1. Open in ChatGPT
2. Copy and paste the prompt from `TASK_PLANNER_PROMPT.txt`  
3. Replace the goal and hit enter
